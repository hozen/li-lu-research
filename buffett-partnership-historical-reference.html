Phase 1 原型 历史参照系 China 2.5 定位

Buffett Partnership Ltd.（1957–1969）是理解 China 2.5 当前位置的唯一完整历史原型。这个参照系的价值在于：它提供了唯一一个完整经历「Phase 1 积累 → For Now 等待 → Phase 2 出击 → Phase 3 持续」四阶段的真实案例。

China 2.5 当前 = Buffett Partnership 1966 年。特征：Pipeline 已建立（31 只裁定完成），三重奏未触发（债券建仓进行中），不焦虑不预测。Buffett 1966 年做了两件事同时进行：① 减少股票仓位（债券建仓）② 继续研究 See's（Pipeline 建设）。

最重要的警示：历史相位图是参照系，不是操作手册。Buffett Partnership 在美国 F2 稳定的环境中运作；China 2.5 在中国 F2 晴雨表「改革开放仍在运作但有摩擦」的环境中运作。两者本质不同。

完整时间线 · 1957–1984

1950–56
Buffett 学习期 — Nebraska 本科毕业，拒绝纽约 MBA，回到奥马哈当股票推销员，同时在 Omaha 教 Graham 的《Principles of Investments》。回 Omaha 而非留在纽约，为后来 19 年搭档芒格埋下伏笔。
1957
Phase 1 启动 — Buffett 以 10.51 万美元创立合伙基金。芒格同年加入。
1957–66
Phase 1 积累期 — 年化回报 23.6%（Dow 同期约 7.1%），超额 16.5 个百分点。规模从 $300,000 → $44,000,000（145 倍）。策略：Cigar-Butt + Generals + Workouts + Controls 四工具并行。
1962
熊市验证年 — Dow -8.7%，BPL +13.9%（超额 +22.6 个百分点）。使用的是 Cigar-Butt（有形资产清算底线），≠ 越跌越买腾讯的理由。参见「1962 年警示」章节。
1966
Phase 1 末期 · For Now 开始 — Burlington Industries IPO 泡沫，F4 信号失真，开始减少股票仓位。Buffett 备忘录：「衡量标准不是市场，是当前的投资环境是否提供了足够的机会。」
1967–71
For Now 等待期 — 清盘合伙基金，持有国债等待。这不是消极等待，而是主动 Pipeline 建设：See's 和 Washington Post 研究多年，1972–1974 年集中兑现。
1969
正式清盘 — 返还所有外部 LP 资金（约 1 亿美元）。清盘理由：规模太大 + 机会质量下降（不是看空市场）。
1972
Phase 2 · See's Candies 收购 — 以约 $3,000 万收购（3.75 倍账面），Buffett 认为「合理价格」。See's 护城河：人情网络（加州送礼文化，50 年提价验证）。50 年后累计为 Buffett 贡献超 $20 亿利润。
1974
Phase 2 · Washington Post lollapalooza 触发 — 能源危机 + 经济衰退 = F4 触发，市值 $0.82 亿，Graham 内在价值 $4 亿 +（4.8 倍安全边际），Buffett 大量买入 $1,000 万，最终回报超 $1 亿。
1974–84
Phase 2 执行期 — See's/WP/BN + Berkshire 整合。规模 $100M → $2.8B（28 倍）。
1985 至今
Phase 3 · 成熟期 — Berkshire 内生增长，Buffett Partnership 历史任务完成。

三大策略工具 · 1957–1966 Phase 1

Cigar-Butt

用 Graham 清算价值折扣买「垃圾公司」。越跌越买，因为有形资产底线支撑。「用 50 美分买一美元。」代表案例：Dempster Mills（最终失败）。

Generals

合理价格买优质公司（被动少数股权）。机会涌现 ↔ F4↑（价格回归正常）；机会枯竭 ↔ F4↓（信号失真）。

Workouts

套利交易（并购/清盘/要约收购事件驱动）。催化剂来自企业层面，基本独立于宏观晴雨表。Buffett 1967–1969 年 F4 失真时增加比重。

Controls

控股型（主动管理，推动清算或重组）。持有 Controls = 对特定制度环境的长期信任。F2 晴雨表强 = 护城河成立；F2 晴雨表弱 = 制度基础动摇。

两种 Phase 2 策略 · See's vs Washington Post

这是最重要的投资认知之一。See's 和 WP 用了完全不同的 Phase 2 策略，不能混用同一逻辑。

See's 模板核心机制

See's 护城河三层不可逆性：① 礼物已送出，情感已传递（接受者无法退回礼品的情感价值）② 50 年情感记忆库存（自我强化）③ 代际传递（家庭/文化仪式中代代相传）。这三个特性使 See's 在 50 年内持续提价而未失去客户——人情网络型护城河的终极测试标准。

Washington Post lollapalooza 机制

1974 年通胀 + 失业率组合 = F4（HC1）触发，能源危机 + 经济衰退 = 系统性错杀。Buffett 在别人恐慌时大量买入，不是因为「捡便宜」，而是因为「护城河完整 + 系统性错杀 + Pipeline 已准备好」。

1962 年熊市警示 · 最常见的框架误用

⚠ 最重要警示：腾讯 ≠ Cigar-Butt

Buffett Partnership 1962 年熊市是最常被误用的历史案例。Buffett 1962 年用的是 Cigar-Butt 策略，核心是有形资产清算价值支撑（净资产 >> 市值），「可以越跌越买」是因为清算价值底线在。

腾讯没有清算价值底线。如果腾讯越跌越买，可能抄到半山腰或接飞刀。Morris For Now 的含义：不卖 ≠ 越跌越买；是「持有 + 等待三重奏触发」。

Cigar-Butt（Graham） 情感护城河（See's） 制度信任护城河（WP）
代表标的 Dempster Mills See's / 腾讯 8.5 Washington Post / 阿里 7
安全边际 清算价值折扣（越跌越买） 护城河耐久性 + 合理价格 lollapalooza + 深度折扣
China 2.5 对应 ⚠️ 排除（无清算价值底线） 腾讯 8.5 > 五粮液 7.5 阿里 7（等待 lollapalooza）

Morris For Now 纪律精确版

For Now 纪律 = sigmoid 函数的精确执行区间，有数学边界，不是模糊等待。

lambda(t) = lambda_max / (1 + exp(-k(t - t_mid)))，当前 lambda ~0.029–0.031（sigmoid 中段偏右）。For Now 窗口仍在 = sigmoid 右侧区间 = Morris 纪律仍在有效执行期。

For Now 的精确含义（比「等待」更具体）：

① 标的仍然有效（腾讯/五粮液的护城河没有消失）② 价格不够好（Graham 折扣不足 + lollapalooza 条件未触发）③ 储备弹药（债券）是最优选择（不持有零回报现金）④ 随时准备窗口关闭时全额出击。

Buffett 1966 年给 China 2.5 的最终启示

Buffett 1966 年做了两件事同时进行：① 减少股票仓位（债券建仓）② 继续研究 See's（Pipeline 建设）。这两件事同时进行，不是先等再研究，而是边等边研究。China 2.5 当前也是如此——债券建仓框架已记录，Pipeline 研究的下一步是 F3LI 四指标的实证数据核实。

正常路径

历史相位图的认知价值与局限

认知价值（为什么有用）

提供了唯一一个完整经历四阶段的真实案例：时间参照（1966 年等了多久？1969–1972 年等了多久？）+ 策略参照（See's 和 WP 的具体操作）+ 情绪参照（Buffett 如何处理「机会不够好」的焦虑）。

认知局限（必须警惕）

① 制度背景不同：Buffett 在美国 F2 稳定环境中；China 2.5 在 F2「改革开放有摩擦」环境中。② 规模差异：BPL 1969 年 $1 亿；China 2.5 潜在规模可能影响市场本身。③ 护城河类型差异：See's 是消费品情感护城河；腾讯是数字基础设施护城河。④ F4 风险不对称：Buffett 面对越南战争+水门；China 2.5 面对中美超级大国博弈风险。

✓ 结论

历史相位图是参照系，不是操作手册。China 2.5 需要建立自己的操作纪律，而不是简单复制 Buffett Partnership 的时间表。
