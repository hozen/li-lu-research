相变点（Phase Transition Point）的精确含义

物理学中，「相变」指的是物质从一种相（固态/液态/气态）切换到另一种相的突变点。在China 2.5框架中：

相（Phase）晴雨表状态Morris衰减率λ置信度D(t)持有期
稳定相（Phase 2.5） F1=F2=F3=0（均未触发） λ ≈ 0 D(t) ≈ D₀（不变量） T = ∞（永久持有）
变革相（Phase 3.0） ∃ci: ci触发 λ >> 0 D(t)快速衰减 T = 有限（框架更新或退出）
核心洞察：China 2.5晴雨表F1/F2/F3 = Morris衰减函数的相变点探测器。晴雨表未触发时，系统处于稳定相（置信度不变量）；晴雨表触发时，系统从稳定相切换到变革相（置信度骤降）。这不是连续衰减，而是相变——从一种稳态到另一种稳态的突变切换。

T(F1, F2, F3) = {

  有限 if ∃ci: ci触发

}
