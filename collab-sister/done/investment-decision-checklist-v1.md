## ⚠️ 前置声明

**此清单不能替代独立思考。** 它是思考的框架，不是思考的替代品。

**使用方法：** 每做一笔投资决策前，完整填写此清单。若任何一步无法通过，决策必须暂停，直到问题被解答或确认为能力圈外。

## 阶段一：能力圈过滤（第零步，必做）

> 能力圈是护城河判断的**前置条件**。不知道自己边界在哪里的"能力圈"，根本不是能力圈，只是自负。

### 1.1 能否预测10-20年自由现金流？

**问题：** 能否以合理确定性预测该公司未来10-20年的自由现金流？

**判断标准：**
- ✅ **能** → 进入1.2
- ❌ **不能** → 能力圈外，**这笔投资需要更严格的安全边际或直接放弃**

**填写：**
```
标的吗名称：_____________________
10-20年FCF预测：_____________________
确定性评估：⬜ 高 / ⬜ 中 / ⬜ 低
理由：_____________________
```

### 1.2 证伪标准是否明确？

**问题：** 如果你的投资论点正确，那么**什么事件或数据出现时，会证明你错了？**

> "When you advance an argument, you must be able to tell me which premises will disprove this argument."
> — Buffett/李录能力圈核心标准

**填写：**
```
投资论点：_____________________
如果以下事件发生，论点失效：
1. _____________________
2. _____________________
3. _____________________
```

**判断标准：**
- ✅ 能明确列出3个以上具体证伪条件 → 通过
- ❌ 无法回答或证伪条件模糊 → 能力圈外或论点不够清晰

### 1.3 能力圈定位

**问题：** 该公司在你的能力圈中处于什么位置？

| 位置 | 定义 | 行动 |
|------|------|------|
| ✅ 圈内 | 深入理解行业，有持续信息获取渠道，能预测10-20年FCF | 按正常流程评估 |
| 🟡 边缘 | 理解行业基本逻辑，但存在关键未知变量 | 降低仓位，提高安全边际 |
| ⚠️ 圈外 | 无法可靠预测FCF，不理解行业核心驱动因素 | 不投资或极小仓位（<3%）|

**填写：**
```
能力圈定位：⬜ 圈内 / ⬜ 边缘 / ⬜ 圈外
关键未知变量：_____________________
```

## 阶段二：护城河评估（通过阶段一后执行）

### 2.1 Morris 四指标维度判断

**对于每个维度，给出 1-5 分（1=极弱，5=极强）**

| Morris 维度 | 评估问题 | 得分 | 关键证据 |
|------------|---------|------|---------|
| **城市化率** | 用户/数据密度是否持续提升，形成网络效应？ | __/5 | |
| **信息能力** | 数据/AI/技术是否形成累积优势壁垒？ | __/5 | |
| **能量获取** | 规模是否带来持续成本或体验优势？ | __/5 | |
| **战争能力** | 是否高度依赖政府授予/行政许可？ | __/5 | |

**Morris 综合评级：**
- 城市化率 + 信息能力 ≥ 8/10 且战争能力 ≤ 2/5 → ✅ 优先候选
- 任一维度 ≥ 4/5 → 该维度形成核心护城河
- 战争能力 ≥ 4/5 → ⚠️ 政府授予型，降权处理

### 2.2 护城河类型判断（最关键一步）

**核心测试：**

> 如果政府明天取消该公司的这项竞争优势，护城河会消失吗？

- ✅ **不会**（市场竞争中建立）→ **市场建立型**
- ⚠️ **会**（依赖政府行政许可）→ **政府授予型**

**填写：**
```
护城河来源：_____________________
如果政府取消这个优势：
⬜ 护城河消失 → 政府授予型 ⚠️
⬜ 护城河不消失 → 市场建立型 ✅
判断理由：_____________________
```

### 2.3 Iron Law 第三层验证

**问题：** 该公司是否正在成为"最大市场 → 唯一市场"？

| Iron Law 层 | 问题 | 判断 |
|------------|------|------|
| 第一层（1+1>2） | 是否比竞争对手更有效利用资源？ | ⬜ 通过 / ⬜ 未通过 |
| 第二层（知识乘数） | 跨学科思维是否产生指数级创新？ | ⬜ 通过 / ⬜ 未通过 |
| **第三层（规模经济）** | **是否成为最大市场且不可逆地向唯一市场演变？** | ⬜ 通过 / ⬜ 未通过 |

**第三层通过标准：**
- 是行业内规模最大
- 规模带来可量化的成本或体验优势
- 竞争格局显示"Largest → Only"趋势

## 阶段三：China 2.5 阶段特殊考量

> 中国2.5阶段熵增压力最大，护城河更稀缺，真正护城河溢价更高。

### 3.1 China 2.5 风险评级

| 风险维度 | 评估内容 | 风险 |
|---------|---------|------|
| 政策风险 | 高度依赖政府授予（采矿权/牌照/频谱）？ | 🟡 中 / ⚠️ 高 |
| 汇率风险 | 持有美元/港元资产面临汇率波动？ | 低 / 🟡 中 / ⚠️ 高 |
| 制度摩擦 | 中国做生意制度成本高？ | 🟡 中 / ⚠️ 高 |
| 接班风险 | 核心管理层离任后护城河可续？ | 低 / 🟡 中 / ⚠️ 高 |

### 3.2 管理层诚信评估（中国2.5核心）

> 中国2.5阶段，管理层诚信是护城河的**前置条件**，不是加分项。

| 评估问题 | 是 ✅ | 否 ⚠️ |
|---------|------|------|
| 10年以上可验证的诚实经营记录？ | ⬜ | ⬜ |
| 财务造假/大股东掏空/不公平关联交易历史？ | ⬜ 无 / ⬜ 有 | |
| 主动披露负面数据（如GEICO费用率跳升时Abel式坦诚）？ | ⬜ | ⬜ |
| 中小股东权益在治理结构中有保障？ | ⬜ | ⬜ |
| 股价下跌时是否出现管理层主动回购？ | ⬜ | ⬜ |

**诚信管理层结论：**
- ✅ 全部通过 → 诚信前置条件满足
- ⚠️ 任何一项存疑 → 提高安全边际要求，缩小仓位
