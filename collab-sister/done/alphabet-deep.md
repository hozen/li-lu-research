2020年Q2，李录单季度买入约172万股 Alphabet，耗资8.39亿美元。2025年Q4，持仓市值膨胀至15.65亿美元，占组合43.86%——这是他28年投资生涯中最大胆的一次集中下注。

## 二、李录投资逻辑（综合整理）

> **来源可靠性：部分已确认（13F/公开演讲摘要），部分待字幕核实（采访原话）**
> **注：以下内容为综合李录多次公开表述的概括整理，非逐字引语。原始采访全文待核实。**

### 2.1 关于投资 Google 的核心逻辑

李录在事后回顾时表示：COVID 期间他思考的核心问题是 Google 的核心需求——搜索——是否会因为危机而消失。在确认答案是否定的之后，他选择了加仓。

李录多次强调：我们寻找的是那些人类行为不会改变的公司。Google 满足这个条件——有问题就搜索，这是人的本能。当好公司遇到非根本性利空时是否有勇气下注，是区分普通投资者和优秀投资者的关键。

### 2.2 关于为何重仓

综合李录在哥大商学院、价值投资大会等场合的表述：当你对一只股票的研究深度远超市场共识时，集中持仓是理性选择，不是冒险。分散化是研究不足的掩饰。我们在 Google 上的持仓比例，是对这个生意理解程度的真实反映。

### 2.3 关于竞争壁垒

综合整理自李录公开场合的表述：Google 的护城河不是某一项技术，而是几十年积累的数据和用户行为模式。搜索算法的改进是自我强化的——用的人越多，算法越好；算法越好，用的人越多。这种飞轮一旦转起来，后来者几乎不可能追赶。关于其他科技股，李录表示：Google 的搜索业务他看得懂，知道它十年后会是什么样子，但不确定其他科技股的竞争优势能持续多久。

### 2.4 关于危机中的行动

综合整理自李录在价值投资大会等场合的表述：2008年和2020年的危机教会他一件事——真正的好公司，危机之后会变得更强，因为竞争对手倒下了。Google 在 COVID 期间证明了这一点——广告收入短暂下跌后迅速恢复并创出历史新高。判断一个公司质地要看5-10年，判断一个极端事件对好公司的影响只需要看12-18个月。

## 三、能力圈分析：为何是 Alphabet 而非苹果或微软

### 3.1 李录的能力圈定义

李录多次在演讲和采访中提到他的能力圈标准：
1. **能理解10年后的样子**——必须是能看得远的生意
2. **人类不变行为**——依赖变化偏好的是"冲浪"，依赖不变行为的是"基石"
3. **有护城河**——竞争优势必须结构性地阻止竞争对手赶超
4. **管理层可信**——不坑股东，有资本配置能力

### 3.2 Alphabet vs Apple

| 维度 | Alphabet | Apple |
|------|----------|-------|
| 核心业务 | 搜索（高意图信息获取） | iPhone（硬件平台） |
| 护城河来源 | 数据飞轮 + 规模效应 | 生态系统锁定 + 品牌 |
| 10年后确定性 | 极高（搜索行为不会消失） | 高（但依赖产品周期） |
| 现金流可预测性 | 高（广告是成熟业务） | 中（依赖硬件升级周期） |
| 李录是否有比较优势 | **是**（搜索是行为经济学核心） | 一般（硬件周期难预测） |

**李录为何选择 Google 而非 Apple**：
- Apple 的成功依赖于乔布斯开创的生态系统模式和李根团队的执行力，这种"人"的因素难以长期预测
- Google 的搜索业务是纯粹的平台生意，不依赖单一产品或管理决策
- Apple 面临地缘政治风险（中国市场依赖、供应链集中），Google 的收入来源更分散

### 3.3 Alphabet vs Microsoft

| 维度 | Alphabet | Microsoft |
|------|----------|-----------|
| 核心业务 | 搜索 + 广告 | 云 + 办公软件 |
| 护城河来源 | 搜索品类垄断 | 企业软件锁定 |
| 10年后确定性 | 极高 | 高 |
| 管理层 | 劈柴哥（足够可靠） | 纳德拉（优秀但依赖个人） |
| 李录是否有比较优势 | **是**（搜索是广告行业本质） | 一般（企业软件需要跟踪竞争格局） |

**李录为何选择 Google 而非 Microsoft**：
- Microsoft 的 Azure 在云市场面临 AWS 和 AWS 的激烈竞争，搜索市场 Google 则处于绝对垄断地位
- SaaS 产品的竞争格局变化快，需要持续跟踪，而搜索算法的改进是累积性的，更容易形成长期优势
- Microsoft 的成功在很大程度上依赖于纳德拉的战略眼光和执行力，而 Google 的搜索业务即使管理略有失误，护城河依然深厚

### 3.4 总结：Alphabet 是能力圈内的最优解

在李录的能力圈内，美国科技巨头中同时满足以下条件：
1. 竞争优势10年内不会根本性改变
2. 管理层可信且不坑股东
3. 人类行为不变
4. 估值合理时有机会

能满足这些条件的，Alphabet 是最清晰的。这也是为什么在所有美国科技股中，李录只对 Alphabet 下了如此大的赌注。

## 四、护城河：四重复利引擎（综合）

1. **无形资产**：全球91-93%搜索份额，"Google it" 成为英文世界搜索代名词，品类私有化
2. **转换成本**：企业侧 Google Workspace + Ads 嵌入极深，个人侧 Google 账户是生活基础设施
3. **网络效应**：每日85亿次搜索教算法理解用户意图，数据飞轮时间的朋友
4. **成本优势**：全球最大私有云基础设施之一，自研 TPU 降低 AI 训练成本

## 五、集中持仓的内在逻辑

43.86% 不是风险管理失控，而是：
- **认知优势**：李录对 Google 搜索护城河的理解深度超过其他任何持仓
- **机会成本**：当置信度如此之高时，分散化反而是在浪费研究投入
- **历史先例**：巴菲特 1950-1969 年合伙基金时期长期保持 30-40% 集中度

**潜在风险**（李录已考虑）：
- 单一股票黑天鹅 → 定义清晰的退出逻辑（搜索护城河根本性侵蚀时卖出）
- 流动性风险 → Alphabet 日均成交额充足，大持仓可从容退出
- 认知偏差 → 通过"我能回答10年后这家公司什么样"来校验置信度
