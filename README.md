## 🛡️ 每日防护检查清单

### 买入前5问
1. **Incentive问**：这笔投资的推荐者有什么激励？（规模佣金/声誉激励驱动？）
2. **Commitment问**：我是否已经在同类标的上有持仓，导致不愿意认错？
3. **Social Proof问**：市场共识是否在看多这个方向？
4. **Deprival问**：我是否因为已经亏损而不愿意止损？
5. **Doubt-Avoidance问**：我是否不愿意质疑当初买入的判断？（Munger Inversion）

### 持仓评估4问（季度/年度）
1. **熵减闭环**：这套机制是否还在自运转？
2. **China 2.5**：制度前提假设是否仍然有效？
3. **买入逻辑**：如果从零开始，我还会买这个标的吗？
4. **反向问**：什么会导致这笔投资失败？（Munger Inversion）

### 卖出调仓3问
1. **新证据**：有新的可证伪条件被触发了吗？
2. **反向答案变化**：买入时的反向问题的答案变了吗？
3. **Pain-avoiding Denial自查**：我是否因为「现实太痛苦所以扭曲」？
