## 🎯 李录投资核心要点

### 三大核心投资原则
1. **所有者心态** - 以100%拥有者角度思考
2. **理性诚实** - 了解自己的能力圈
3. **深入了解企业** - 比管理层更懂

### 投资标准
- 持久护城河 (Durable Moat)
- 高资本回报率 (可持续)
- 以低于内在价值买入 (安全边际)
- 长期持有，在下跌时加仓

### 关键洞见
> "如果你只是假装了解企业，你会在困难时期被考验"

> "长期投资回报与公司资本回报率完美相关"

> "每5-10年发生一次百年一遇的危机"
