波普尔批判理性主义

卡尔·波普尔（1902–1994）的批判理性主义，是 China 2.5 框架的哲学地基。它解释了为什么 Morris「For Now」、Buffett Partnership 费率结构和晴雨表体系，三者独立收敛于相同的认识论纪律。

哲学来源 · 可证伪性原则 · 四重框架之一

一、为什么投资者需要波普尔？

价值投资的核心挑战不是「找到好公司」，而是「知道自己在什么时候犯错」。波普尔为这个问题提供了最精确的哲学答案。

三种投资失败的根源

  • 预测失败：宏观预测、地缘政治预测、短期股价预测——这些本质上不可证伪，因此本质上不是知识
  • 框架失效：能力圈边界内，但框架本身的前提条件发生了变化
  • 行为偏差：知道应该做什么，但情绪驱动作出相反决策

波普尔的批判理性主义同时解决这三个问题：

可证伪性原则自动排除不可证伪的预测（宏观/地缘政治/时机）——这些不是知识
预先声明可证伪条件，使得框架失效可以被客观检测——框架在数据触发前始终成立
预设触发规则，消除了情绪决策的空间——行为偏差的制度化解决方案

二、批判理性主义核心原则

可证伪性原则（Falsifiability Criterion）

「一个理论如果是科学的，就必须能够被经验检验，并且有可能被证明是错的。」
— 卡尔·波普尔《科学发现的逻辑》（1934）

可证伪 ≠ 被证伪。 China 2.5 框架至今未被证伪，但这不是问题——关键是它在原则上可以被证伪。

China 2.5 框架的可证伪条件

晴雨表 可证伪条件（预设触发规则） 证伪成功含义
F1 微信 DAU 较峰值下跌 ≥ 20% 且替代渗透率 > 30% 平台护城河的根本性断裂
F2 PCAOB 两年无进展 或 美国国会通过中概股退市法案 制度改善停滞或逆转
F3 阿里云增速 < 5% 且 腾讯广告/金融科技毛利率连续两季度下降 能量悖论转型路径受阻

证伪失败 ≠ 证实

波普尔最深刻的洞察：证实在逻辑上不可能，但证伪在逻辑上可以一次完成。

「无论多少只白天鹅都不能证明'所有天鹅都是白的'，但一只黑天鹅就能证伪它。」
— 波普尔，黑天鹅问题

应用到 China 2.5：无论多少次财报数据符合预期，都不能「证明」框架成立；但只要 F1/F2/F3 中任何一个触发，框架就需要修正或退出。

批判理性主义三层结构

第一层：猜测

形成理论或假设（China 2.5 框架）

第二层：反驳

设计尽可能严格的检验（晴雨表 F1/F2/F3）

第三层：淘汰

如果检验失败，修正或放弃理论

三、三角闭环：波普尔 × Morris「For Now」× Buffett Partnership

三个独立来源，在没有任何相互引用的情况下，收敛于完全相同的认识论纪律。这是多独立构造器系统（Multiple Independent Constructors）的最强证据。

波普尔（1934）：哲学来源

批判理性主义：科学知识通过证伪失败迭代，而非证实积累。

Morris「For Now」（2010 论文）：操作化

五步操作化（Piecewise Decision Function）: D(E; θ) = A₀ if ∀ci: ci 未触发 → 继续持有 A₁ if F1 触发 → 减仓 A₂ if F2 触发 → 减仓 A₃ if F3 触发 → 减仓 A₃ if ∀ci: 全部触发 → 退出

Buffett Partnership 费率结构（1956–1969）：制度化

Buffett Partnership 费率结构 波普尔/Morris 含义
6% 门槛收益率 预先声明知识边界（能力圈之外的收益不参与分成）
25% 激励费 利益与批判性检验绑定（LP 和 GP 对齐）
无固定管理费 承认错误的经济成本内部化（没有旱涝保收）
LP 随时退出权 同行评审制度化（LP 可以用脚投票）

三角闭环的数学统一性

三者共享的数学结构：布尔决策函数 + 预设触发条件 + 预先声明知识边界，而非概率优化。

四、波普尔 vs 贝叶斯：为什么 China 2.5 不是贝叶斯方法

维度 波普尔/Morris「For Now」 贝叶斯方法
置信度来源 独立证伪检验（外生触发） 连续更新后验概率（内生计算）
置信度随时间 不变量（晴雨表未触发时不变） 连续变化（每次新数据都更新）
决策函数 布尔（0/1 触发） 概率加权（连续值）
预设检验条件 预先声明，不随数据修改 可以随新证据调整
过度拟合风险 无（预设条件无法事后修改） 存在（持续更新可能导致）
重要区分：Morris「For Now」本质不是贝叶斯方法。原因一：不更新置信度，只执行预设规则；原因二：预设检验条件，拒绝数据挖掘；原因三：承认「零贝叶斯更新」的合理性（持续未触发 ≠ 越来越确定）。

五、Morris 衰减函数 × 晴雨表 = 相变点探测器

Morris「For Now」2010 论文原版提出：预测有效性随时间尺度衰减（内生衰减函数 D(t) = D₀ × e−λt）。China 2.5 晴雨表体系与此存在深层整合：

整合命题： λ（衰减率）= f(F3 晴雨表状态) = 0 if F3 未触发（Phase 2.5 稳定期） = 快速增加 if F3 触发（Phase 3.0 转型期） 持有期 T(F1, F2, F3) = ∞ if F1=F2=F3=0（稳定相，置信度不变量） 有限 if ∃ci: ci 触发（变革相，框架修正或退出）

晴雨表 = 相变点探测器（Phase Transition Point Detector）

晴雨表未触发

稳定相（Stable Phase）：λ ≈ 0，置信度不变量，无限持有期

晴雨表触发

变革相（Transformative Phase）：λ >> 0，置信度骤降，框架更新或退出

Buffett 1969 年清盘的本质：识别到 λ 正从 ≈0 急剧增加到 >>0（越战升级/布雷顿森林体系崩溃/利率飙升）。这不是「置信度随时间自然衰减」，而是「结构性相变点被触发，持有期从无限变为有限」。

六、波普尔认识论与 China 2.5 置信度

China 2.5 置信度 4.88/5 的精确含义

置信度 4.88/5 不是概率陈述（贝叶斯后验概率），而是三层含义的复合：

层次 含义 数学性质
波普尔确认度 在足够多独立证伪尝试均失败后，框架的确认度为高 B(H, e) 确认度（非概率）
决策理论 EU（继续持有）> EU（放弃框架） 预期效用最大化陈述
Morris 操作化 晴雨表 F1/F2/F3 均未触发 → 继续持有 分段布尔决策函数
精确纠正：「证伪失败 → 置信度 +ε」是不精确的近似表述。波普尔的 B(H, e) 不是概率（不满足可列可加测度），不能说 4.88 + 0.01 = 4.89。正确表述：「证伪失败 → 确认度 B(H, e) 维持高位 → 继续坚持 H 是理性决策（预期效用最大化）」。

波普尔的「我知道我不知道」

波普尔区分三类知识：

  • 第一类：我知道我知道的
  • 第二类：我知道我不知道的（波普尔的认识论核心）
  • 第三类：我不知道我不知道的

Buffett Partnership 1969 年清盘时说的「我知道没有足够好的标的」——这是第二类知识的完美操作化（第一人称确定声明）。China 2.5 的三重预先声明（不知道中国 AI 革命何时/不知道 F2 改善何时/不知道腾讯 AI 商业化何时规模化）同样如此。

七、05-13 晴雨表窗口期：波普尔认识论的操作检验

2026 年 5 月 13 日 19:30，腾讯 Q4+Q1 财报 + 阿里 Q4+Q1 财报发布，这是 China 2.5 框架的首次晴雨表完整检验。

三重晴雨表的布尔逻辑

检验结果（05-13 19:30）: [F1=0 ∧ F2=0 ∧ F3=0] → 继续持有（证伪失败） [F1=1] → F1 触发（证伪成功）→ 减仓 [F2=1] → F2 触发（证伪成功）→ 减仓 [F3=1] → F3 触发（证伪成功）→ 减仓 [F1=1 ∧ F2=1 ∧ F3=1] → 三重触发 → 退出 F1/F2/F3 = 0 表示未触发，= 1 表示触发。 晴雨表只有 0/1 两种状态，不存在概率加权。

波普尔视角下的晴雨表

晴雨表是预设的严格检验，不是事后选择的数据
F1/F2/F3 任一触发 = 证伪成功 = 框架修正或退出
晴雨表未触发 = 证伪失败 = 确认度维持高位
检验标准在数据发布前预先声明（避免后见之明偏差）
