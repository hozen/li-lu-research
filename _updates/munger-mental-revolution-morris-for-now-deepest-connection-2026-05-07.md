## 二、Munger 心理革命的精确内容

### 2.1 心理革命 vs 智识革命

Munger 在 USC 1994 演讲中明确区分：

- **智识革命（Intellectual Revolution）**: 学习新知识、新模型、新事实
- **心理革命（Mental Revolution）**: 改变你如何处理已有知识的方式

**投资关键**：大多数投资者的缺陷不是知识不足，而是**处理知识的方式有系统偏差**。

### 2.2 心理革命的三个核心机制

**机制一：自动运行的偏见链条**

```
刺激输入 → 偏见自动激活 → 行为输出  [无意识介入]
刺激输入 → 偏见激活 → [芒格打断器] → 冷静评估 → 行为输出  [正确流程]
```

Morris「For Now」= 预设证伪条件 = 在数据刺激到达之前，在心理链条启动之前就打断。

**机制二：多模型联合作战**

单一模型 → 工具谬误（用锤子看所有问题）
多模型 → 意识调用 → 选择正确模型
内化多模型 → 无意识调用 → 接近芒格式智慧

Morris「For Now」要求预先声明证伪条件——这需要多个模型的联合作战（历史/哲学/经济学/心理学同时输入）。

**机制三：逆向思维作为默认设置**

芒格：聪明人最大的智慧是倒过来想（Inversion）。
Morris「For Now」：倒过来想——如果我错了，数据会从哪里出现？

不是"我希望我是对的"，而是"如果我错了，什么数据会证明"。

## 三、Morris「For Now」的心理学维度

### 3.1 为什么 Morris「For Now」不只是认识论

现有分析将 Morris「For Now」定位为认识论工具（承认预测不可靠）。这是对的，但不完整。

**Morris「For Now」同时是心理工程工具**：

- 预设证伪条件 → 减少事后合理化（post-hoc rationalization）
- 等数据来了再说 → 减少行动焦虑
- 不焦虑/不预测/不行动 → 减少情绪耗损
- 专注研究/持续监控 → 将注意力导向正确方向

### 3.2 Morris「For Now」在心理学上的精确对应

| Morris「For Now」组件 | 芒格心理学对应 |
|---------------------|-------------|
| 预设证伪条件 | 避免"确认偏见"自动运行链条 |
| 等数据来了再说 | 避免"行动偏误"（action bias）|
| 不焦虑 | 避免"损失厌恶"主导决策 |
| 不预测 | 避免"过度自信"偏差 |
| 持续监控 | 正确版本的"可得性启发"（定期更新，而非只记忆极端事件）|

### 3.3 Morris「For Now」的心理脆弱性

**最危险的误用**：将 Morris「For Now」用于**延迟决策**（indecision）而非**等待正确数据**（disciplined patience）。

两者外观相似，但：
- 延迟决策 = 心理偏见（损失厌恶+行动恐惧）
- 等待正确数据 = 心理革命（纪律+耐心）

### 5.2 为什么框架完整版比简化版更有力

**简化版**（常见误用）：
Morris「For Now」= 什么都不做 / 等着 / 保守

**完整版**（正确理解）：
Morris「For Now」= 心理革命（预先安装）+ 认识论谦逊（承认预测不可靠）+ 行动纪律（预设触发条件）+ 执行意愿（条件触发时行动）

简化版的问题：等待变成拖延，不行动变成不思考。

完整版的优势：**等待是有结构的，不行动是有条件的**。
