## 一、PTF函数完整数学结构

### 1.1 PTF原始定义

**Morris (2011) 整体生产率函数：**

```
PTF(t) = [N(t) × e(t)] × [A(t) × μ(t)] × [1/ECoE(t)]
```

**四项分解：**

| 项 | 符号 | 含义 | 单位 | 历史趋势 |
|----|------|------|------|---------|
| **Population** | N(t) | 人口数量 | 百万人 | 农业革命后快速增长，工业革命后加速 |
| **Energy per capita** | e(t) | 人均能量获取 | kcal/人/天 | 工业革命后爆发式增长 |
| **Organization** | A(t) | 组织效率指数 | 无量纲 | 制度复杂度提升 |
| **Knowledge** | μ(t) | 认知效率 | 无量纲 | 信息革命后大幅改善 |
| **ECoE** | ECoE(t) | 能量获取成本 | 能量单位/能量单位 | 工业革命后持续下降 |

### 1.2 PTF函数的物理含义

PTF函数测量的是**单位能量投入所产生的社会发展成就**。

- **PTF ↑**：在给定能量投入下，社会能维持更多人口、更高的组织复杂性、更大的知识积累
- **PTF ↓**：能量获取成本上升或组织效率下降，单位能量产生的社会发展成就减少
- **Morris For Now窗口**：PTF在正值区间稳定增长的历史时期

**历史PTF值：**
- 公元前10000年（农业革命前）：PTF ≈ 1（基准）
- 公元前3000年（城市文明）：PTF ≈ 10
- 1776年（工业革命）：PTF ≈ 100
- 2000年（信息革命）：PTF ≈ 10,000
- 2026年（AI时代）：PTF ≈ 15,000-20,000（估算）

### 1.3 PTF的对数形式

Morris选择用**对数形式**表达PTF，因为这揭示了增长是乘法而非加法的：

```
log(PTF) = log(N) + log(e) + log(A) + log(μ) − log(ECoE)
```

**对数形式的含义：**
- 每一项的增长贡献是**叠加的**
- 人口增长10倍 = 知识效率提升10倍（在对数尺度上等价）
- 这解释了为什么工业革命后PTF增长是加速度式的，而非线性的

## 二、Lambda分解：PTF增长率的五层分解

### 2.1 Lambda定义

```
λ(t) = d(PTF)/dt / PTF(t) = d(log PTF)/dt
```

λ(t)是PTF的瞬时增长率，测量PTF变化的"加速度"。

**展开：**

```
λ(t) = λN(t) + λe(t) + λA(t) + λμ(t) − λECoE(t)
```

| 分量 | 定义 | China 2.5当前值 | 说明 |
|------|------|---------------|------|
| **λN** | 人口增长率 | ≈0（负增长） | 人口老龄化压制 |
| **λe** | 人均能量增长率 | ≈+0.01~0.02 | HC1新能源转型 |
| **λA** | 组织效率增长率 | ≈+0.005~0.01 | HC2制度改革 |
| **λμ** | 认知效率增长率 | ≈+0.02~0.05 | HC4 DeepSeek R1 |
| **λECoE** | ECoE变化率 | ≈−0.01~−0.02（ECoE下降） | 能量成本下降 |
| **λ(t) 合计** | — | **≈0.025~0.045** | sigmoid中段 |

**λ(t)的三个区域：**
- **λ > 0**：PTF扩张，Morris For Now窗口仍在
- **λ = 0**：PTF停滞，Hard Ceiling触及
- **λ < 0**：PTF收缩，Iron Law螺旋失控

### 2.2 Morris For Now窗口的λ定位

Morris For Now窗口（~1850-2150年）在λ维度上的位置：

```
sigmoid函数：λ(t) = λ_max / (1 + e^(−k(t − t_mid)))
```

| 参数 | 数值 | 说明 |
|------|------|------|
| **t_mid** | ≈2050年 | sigmoid中点 |
| **λ_max** | ≈0.05 | PTF最大增长率（工业革命高峰期） |
| **k** | ≈0.03 | sigmoid陡峭度参数 |
| **当前t（2026）** | t_mid − 24年 | sigmoid后半场初期 |

**关键洞见：** 当前China 2.5的λ ≈ 0.029，**远低于**1850-1950年的λ ≈ 0.050，但**仍为正值**。这意味着Morris For Now窗口仍在，但已从"加速期"进入"减速期"——这正是Buffett Partnership 1970-1973年所处的历史区间。

## 三、ECoE三层历史转型

### 3.1 ECoE定义

**Energy Cost of Energy（能量获取成本）：**

```
ECoE = 获取一单位可用能量所需的能量单位
```

ECoE越低，净剩余能量越多，社会可维持更高的组织复杂性。

**ECoE的历史三次跃迁：**

| 时代 | ECoE范围 | 净剩余 | 社会形态 |
|------|---------|-------|---------|
| 觅食时代 | 5-10 | 极低（<10%） | 游群（15-30人） |
| 农业时代 | 1-3 | 中等（30-50%） | 城市国家 |
| 化石燃料时代 | 0.05-0.1 | 极高（>90%） | 工业文明/民族国家 |

**ECoE与PTF的关系：**
- ECoE ↓ → PTF ↑（在给定N×e×A×μ下）
- ECoE ↑ → PTF ↓（净剩余减少）
- ECoE是PTF的"阻力项"

### 3.2 ECoE三层分析框架

**第一层：物理能量ECoE（HC1）**
- 化石燃料开采效率
- 新能源（光伏/风电/储能）成本曲线
- China 2.5当前：光伏LCOE降至0.20-0.30元/kWh（历史性低点），HC1 ECoE仍在下降通道

**第二层：政治经济学ECoE（HC2）**
- 制度改革成本（U*(t)动态调整的能量成本）
- 交易成本：分散决策的协调成本
- China 2.5当前：改革开放降低了HC2 ECoE（三中全会机制仍在运作）

**第三层：社会资本ECoE（HC3）**
- 信任网络维护成本
- 信息传递效率
- China 2.5当前：微信/支付宝降低社会资本ECoE（Ostrom退出选项）

**第四层：认知接口ECoE（HC4）——历史上首次**
- 知识获取成本（μ的倒数）
- DeepSeek R1大幅降低AI使用门槛
- China 2.5当前：DeepSeek R1/O1使认知ECoE骤降（类比工业革命首次降低物理ECoE）

### 3.3 Japan 1990s ECoE锁定

**Japan 1990s三层ECoE同时锁定：**
- HC1 ECoE：石油危机后日本完全依赖进口化石燃料，ECoE骤升且无法自主
- HC2 ECoE：大藏省精英共识锁定制度改革空间，U*(t)完全固定
- HC3 ECoE：终身雇用制瓦解，信任网络崩溃
- **结果：** 30年Iron Law螺旋，λ始终为正值但极低（≈0.005），PTF增长停滞

**China 2.5 vs Japan：** China有HC4（DeepSeek）作为历史上首次的认知ECoE降低机制，Japan完全没有。

## 五、三层Hard Ceiling × PTF四项映射

### 5.1 Hard Ceiling定义

**Morris Hard Ceiling（铁律上限）：** 文明在接近物理/制度/社会极限时，维护PTF增长所需的能量成本超过可用能量，Iron Law螺旋触发。

**Hard Ceiling三层结构：**

| 层 | 内容 | PTF对应项 | 触发条件 |
|----|------|----------|---------|
| **第一层（物理）** | 能量获取上限 | N×e（人口×人均能量） | 化石能源枯竭或ECoE超临界 |
| **第二层（政治经济学）** | U*(t)制度锁定 | A（组织效率） | 制度改革空间关闭，利益集团捕获 |
| **第三层（社会资本）** | 信任网络耗散 | A×μ（组织×知识） | 社会凝聚力崩溃，信息传递失真 |

### 5.2 PTF与Hard Ceiling的数学关系

**Iron Law螺旋的PTF表达：**

当Hard Ceiling触及：
```
dPTF/dt < 0（PTF收缩）
    ↓
净剩余减少
    ↓
护城河建设资本减少
    ↓
护城河无法维护
    ↓
单位能量产出下降
    ↓
dPTF/dt 更负
    ↓（循环加速）
```

**关键区分：**
- **线性衰退**：dPTF/dt = −k（每次相同幅度下降）
- **Iron Law螺旋**：dPTF/dt = −k·PTF（PTF越高，下降越快）——这才是"Spiral Out of Control"的数学含义

### 10.1 三层时间尺度精确分层

**第一层：晴雨表（季度尺度）**

- 反映Morris For Now窗口位置
