## 一、PTF函数定义与四项含义

### 1.1 PTF函数定义

**PTF（Phase Transition Function）** 出自Morris《战争！》（2016），是Morris SDI的数学重构：

```
PTF(t) = N(t) × e(t) × A(t) × μ(t)
```

**四项完整定义：**

| 项 | 变量名 | 含义 | 单位 |
|----|--------|------|------|
| N(t) | 人口规模 | 人口数量 | 人数 |
| e(t) | 人均能量获取 | 每人每年获取的能量 | J/人均/年 |
| A(t) | 组织复杂性 | 制度/社会结构复杂度（× U*(t)权重） | 无量纲指数 |
| μ(t) | 知识效率 | 单位知识获取成本的倒数（认知效率） | 无量纲指数 |

**PTF物理含义：**
- PTF = 文明的社会发展函数
- PTF越高 → 社会可维持更复杂的组织（城市、国家、贸易网络）
- PTF下降 → 组织复杂性开始崩溃（Rome、MING王朝的历史原型）

**ECoE（Energy Cost of Energy）与PTF的关系：**

```
ECoE(t) = 1 / [e(t) × μ(t)] × 常数
```

- ECoE = 获取一单位可用能量所需的能量单位
- ECoE越低，净剩余能量越多，文明可维持更高组织复杂性
- ECoE与PTF呈反向关系：ECoE↑ → PTF↓；ECoE↓ → PTF↑

### 1.2 三次能量体制与ECoE转型

| 体制 | ECoE范围 | 能量获取特点 | Morris SDI含义 |
|------|---------|------------|--------------|
| 觅食体制 | 5~10 | 95%能量用于获取食物，几乎无剩余 | 人口稳定，组织简单 |
| 农业体制 | 1~3 | 能量剩余开始积累，支撑城市 | 人口增长，城市出现 |
| 化石燃料 | 0.05~0.1 | 工业革命爆发，净剩余骤增 | 组织复杂性指数级增长 |

**ECoE转型机制：**
- 农业革命：ECoE从5-10降至1-3（能量革命）
- 工业革命：ECoE从1-3骤降至0.05-0.1（工业革命本质）
- 新能源转型（HC1）：ECoE继续下降（光伏/风电边际成本趋零）
- AI认知革命（HC4）：ECoE(认知)大幅下降（推理成本骤降）

## 二、PTF lambda（增长率）分解

### 2.1 Lambda定义

```
PTF(t) = N(t) × e(t) × A(t) × μ(t)
λ(t) = dPTF/dt / PTF = λN + λe + λA + λμ
```

**lambda五分量：**
- λN = dN/dt / N（人口增长率）
- λe = de/dt / e（人均能量获取变化率）
- λA = dA/dt / A（组织复杂性变化率）
- λμ = dμ/dt / μ（知识效率变化率）

### 2.2 Lambda的实际含义

| λ(t)区间 | 含义 | Morris诊断 |
|---------|------|-----------|
| λ > 0 | PTF扩张，社会发展仍有空间 | Morris For Now窗口开放 |
| λ ≈ 0 | PTF停滞，社会发展触顶 | Hard Ceiling逼近 |
| λ < 0 | PTF收缩，社会发展倒退 | Iron Law螺旋失控 |

### 2.3 Morris For Now窗口的Sigmoid定位

```
λ(t) = λ_max / (1 + e^(-k(t-t_mid)))
```

**参数估计：**
- λ_max ≈ 0.050（1850-1950年峰值，工业革命红利期）
- t_mid ≈ 2050（Morris For Now中点）
- k ≈ 0.07（sigmoid曲线陡度参数）

**Morris For Now窗口分期：**
- 前半场（1850-2050）：λ加速上升（改革红利释放期）
- 后半场（2050-2150）：λ减速上升（边际收益递减）
- 当前2026年：t = t_mid - 24年，λ ≈ 0.029（sigmoid后半段初期，低lambda稳定区）

**Sigmoid四性质：**
1. λ一阶导（dλ/dt）：变化率的变化，加速度概念
2. λ二阶导：sigmoid拐点附近dλ/dt最大
3. 极限行为：t→-∞时λ→0；t→+∞时λ→λ_max
4. 中点特殊性：t_mid处dλ/dt=0

**Morris For Now的sigmoid解读：**
- 1850-1950 = sigmoid前半场（λ加速，现代化红利爆发）
- 1950-2050 = sigmoid后半场前半段（λ缓慢下降但仍为正）
- 2050-2150 = sigmoid后半场后半段（λ继续下降趋近λ_max）
- 当前2026年 ≈ sigmoid中段偏下（λ为正但递减，For Now窗口仍在）
