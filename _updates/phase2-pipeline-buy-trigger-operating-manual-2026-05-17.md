## 二、Phase 2买入触发条件精确化（终态）

### 2.1 Morris晴雨表三重奏精确条件

**F4↓（低估机会信号）**：
- HC1能量获取成本骤升（类似1974年石油危机）
- Buffett Indicator < 80%（战后最低区间）
- 市场恐慌情绪达到极端（媒体充斥负面情绪）

**F2稳定（政治经济学稳定性）**：
- Popper-Tideman-Functional Unanimity三维联合中≥2维正常
- Tideman委托费合理性改革开放仍在进行
- 三中全会机制产生正向立宪层修正

**F3稳定（社会资本未耗散）**：
- Ostrom四原则中≥3项运作
- 数字平台Ostrom退出选项存在
- 社会信任网络未崩溃

**三条件同时满足 = Morris晴雨表三重奏触发 = Morris For Now窗口内的买入信号**

### 2.2 具体标的Trigger条件（三重验证）

**第一重：Morris晴雨表三重奏满足**
- F4↓ ✅
- F2稳定 ✅
- F3稳定 ✅

**第二重：具体标的See's模板评分≥7/10**
| 评分区间 | 操作含义 |
|---------|---------|
| ≥8.5/10 | 立即观察，合理价格即买入 |
| 7-8.4/10 | 等待F4触发或深度折扣 |
| 6-6.9/10 | 需要额外折扣作为安全边际补偿 |
| <6/10 | 等待，不作为Phase 2标的 |

**第三重：Graham安全边际折扣≥25%**
- 清算价值折扣≥40%：极度低估，优先买入
- 盈利收益率 vs 国债收益率：E/P > 国债收益率+3%
- PE vs 历史均值：PE < 历史均值30%+

**三条件同时满足 = 执行Phase 2买入**

## 三、等待期的正确行动：Buffett Partnership 1967-1971年精确镜像

### 3.1 Buffett Partnership等待期的行动

Buffett在1967-1971年等待期（市场不便宜但晴雨表三重奏未触发）做的事情：

**1967年**：
- 识别Workout机会（短期确定性）
- 持续研究Phase 2候选（Dempster Mill Workout转Control）
- 清盘部分Generals（市场偏贵）

**1968年**：
- Dempster Mill Control触发（清算价值>市值+无力自改）
- 出售Dempster（盈利）
- 持续研究See's

**1969年**：
- 市场极端高估（Buffett Indicator创历史新高）
- 清盘全部Buffett Partnership
- 锁定Phase 1利润，为1974年F4触发准备弹药

**1970-1971年**：
- 持有现金，等待F4触发
- 持续研究Phase 2候选
- 拒绝在市场偏贵时执行Generals

## 五、等待期行动纪律（Buffett Partnership原则）

### 5.1 每日行动清单

1. **监测Morris晴雨表三重奏**（每cron job检查一次）
   - F4是否触发？（HC1成本变化）
   - F2是否稳定？（Popper-Tideman-FU三维）
   - F3是否稳定？（Ostrom四原则）
   - 答案：三重奏未触发 → 精确等待

2. **不主动加仓**（除非三重奏满足）
   - 不预测时机
   - 不在市场偏贵时追高
   - 只在机会出现时识别并执行

3. **持续Pipeline建设**（每轮新增1-2只裁定）
   - 识别新候选
   - 完成See's模板四维度评分
   - 记录于phase2-pipeline-complete-integration-2026-05-17.md

4. **Git同步纪律**
   - 每次裁定完成后commit+push
   - origin/dev=working tree clean
   - origin/master同步（生产环境）
