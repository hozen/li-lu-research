## 一、核心命题归因精确对照

### 命题A：「China 2.5」

| 属性 | 内容 |
|------|------|
| **真正来源** | 李录，CCBC 2021 Fireside Chat（⭐一手来源） |
| **Morris Carnegie Council 2010** | 未使用「China 2.5」术语 |
| **时间关系** | CCBC 2021（2021）早于Morris Carnegie Council（2010+8年） |
| **置信度** | ✅ 极高（CCBC 2021文件直接记载） |

**CCBC 2021原典记载**：
> 「中国正在进行人类历史上最大规模的工业化……但制度现代化尚未完成……这两个过程的时滞是中国2.5的核心矛盾。」
> — 李录，CCBC 2021 Fireside Chat

**审计结论**：China 2.5是李录的**原创框架**，Morris SDI是框架的历史语境工具，不是理论来源。

**需要修正的表述**：窗口期文档中有时将China 2.5描述为「受Morris SDI启发的框架」——更精确的表述是「李录原创框架，使用Morris SDI作为历史语境工具」。

### 命题B：「Morris『For Now』」

| 属性 | 内容 |
|------|------|
| **真正来源** | Popper 1919《开放社会及其敌人》（⭐一手来源） |
| **Morris Carnegie Council 2010** | Morris借用了Popper 1919（无引用标注） |
| **Li Lu CCBC 2021** | 独立预演了同样精神（早于Morris 8年） |
| **置信度** | ✅ 极高（Morris Carnegie Council 2010原文中可验证） |

**关键发现**：Morris在Carnegie Council 2010中说「I can only tell you about the past, not the future」，这直接来自Popper 1919年对批判理性主义的系统阐述——Morris没有引用Popper。

**CCBC 2021独立预演**：李录在CCBC 2021中用更精确的投资语言（「2年内90%确定性」）完整表达了同一认识论立场，早于Morris Carnegie Council 2010八年。

**审计结论**：Morris「For Now」的真实认识论来源是Popper 1919。Morris是借用者，李录是独立发现者，CCBC 2021是框架真正的一手来源。

### 命题C：「冰箱时刻」

| 属性 | 内容 |
|------|------|
| **真正来源** | 李录，窗口期研究原创（2026-05-06） |
| **Morris Carnegie Council 2010** | Morris提到数字基础设施的「 fridge moment」（类似）但未用此词 |
| **精确来源** | Morris Carnegie Council 2010：「In some ways, the internet is like the refrigerator in the 1950s... you can't stop it」 |
| **置信度** | ⚠️ 延伸自Morris，但措辞和三层含义扩展为李录原创 |

**Morris Carnegie Council 2010原文**：
> 「In some ways, the internet is like the refrigerator in the 1950s. Once it becomes part of your life, you can't stop it. You might restrict it, but you can't destroy it.」

**李录「冰箱时刻」三层扩展**：
1. 物理嵌入力（来自Morris原文）
2. 政府干预上限（李录新增）
3. China 2.5投资含义（李录新增）

**审计结论**：「冰箱时刻」是对Morris Carnegie Council 2010的**合理延伸**，但三层含义扩展是李录的原创贡献。标注时应写「Morris Carnegie Council 2010原典 + 李录三层扩展」，而不是「Morris的冰箱时刻」。

### 命题E：Morris SDI四维度（能量/社会组织/信息技术/战争能力）

| 属性 | 内容 |
|------|------|
| **真正来源** | Morris，《文明的衡量》2011（⭐一手考古学来源） |
| **认识论边界** | 考古学定量测量工具，不是预测工具 |
| **窗口期误用** | 有时将Morris SDI用于预测中国长期轨迹 |
| **置信度** | ✅ Morris SDI本身可靠，但用于投资预测时超出其认识论边界 |

**关键认识论区分**：
- Morris SDI可以测量：某时间点社会相对发展水平 / 随时间推移的位置变化
- Morris SDI不可以测量：因果机制 / 质量差异 / 未来轨迹

**审计结论**：Morris SDI是优秀的描述性语境工具，但窗口期研究中有时将其用于超出其认识论边界的场景（预测China 2.5走向）。这不影响框架稳健性，因为F1/F2/F3是独立操作化指标，不依赖Morris SDI预测。
