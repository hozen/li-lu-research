- **关键**：Buffett回 Omaha 而非留在纽约，这选择对他后来19年搭档芒格至关重要

**1960年Buffett备忘录（最关键文件）**：

这是Morris For Now纪律的Buffett原型表述：衡量标准不是市场，是**当前的投资环境是否提供了足够的机会**。

- Partnership合约9%门槛：LP拿走超过9%的部分，这是Buffett的激励约束

## 二、Buffett Partnership三类投资 × Morris晴雨表三层映射（精确版）

### Generals（低估类）= 主要依赖F4晴雨表
```
Generals机会涌现 ← F4↑（价格回归正常）
Generals机会枯竭 ← F4↓（价格信号失真）
Generals改进型 ← F4↓ + F2/F3安全（质量+低估双重条件）
```
**1967-1969年决策**：增加Workouts，减少Generals
**底层逻辑**：F4晴雨表开始失真（Burlington Industries IPO泡沫 = 错误信号），Workouts独立于F4运行

### Workouts（套利类）= 基本独立于晴雨表三重奏
```
催化剂来源：企业行动（并购/分拆/清算/要约收购）
与Dow相关性：极低
Buffett原话："we will look extremely good if it turns out to be a declining year for the Dow"
```
**Buffett 1967-1969增加Workouts比重的底层逻辑**：
- F4失真时，Workouts不受影响（催化剂来自企业层面，不是宏观层面）
- 这是 Buff / Morris晴雨表系统第一次被明确区分

### Controls（控股类）= 对应Morris For Now制度层（F2）
```
持有Controls = 对特定制度环境的长期信任
F2晴雨表强 = Controls护城河成立
F2晴雨表弱 = Controls制度基础动摇
```

### See's/Washington Post模板 = Phase 2的核心标的原型
**See's Candy**：
- 护城河来源：人情网络（社区糖果店，顾客几十年关系）
- 定价权：情感绑定（"感恩节买See's是家庭传统"）
- 转换成本：极高（情感替代品几乎没有）
- **Buffett至今持有**：说明这不是"一次性机会"，而是长期护城河

**Washington Post**：
- 护城河来源：政府关系网络（联邦政府雇员订阅+广告主）
- 1974年买入：lollapalooza条件满足（F4触发 + F2/F3安全）
- 2014年清仓：互联网破坏了F2晴雨表（在线分类广告替代）
- **关键：护城河有制度性依赖，当制度变化时必须重新评估**

**See's/Washington Post的共同特征**：
1. 护城河来自**人际信任网络**（不是规模效应）
2. 转换成本极高（情感锁定或政府关系）
3. Graham内在价值显著低估（Phase 2买入时点）
4. 管理层有股东心态（See's家族愿意谈判，Graham家族愿意接受建议）

1. F4信号失真（Burlington Industries IPO泡沫 = 错误信号）
