### Step 2：数据收集（固定顺序，不可打乱）

**收集顺序**（按重要性排列）：

#### 2.1 F1 = FBS毛利率（权重 >25%）— 最重要

**标准**：
- 基准线：55.7%（Q4 2025）
- 触发条件：F1 < 52.7%（下降超过3个百分点）
- 数据来源：腾讯季度财报 → 营收 ÷ 毛利

**收集操作**：
```
毛利率 = (毛利 ÷ 营收) × 100%
若毛利率 < 52.7% → F1 触发
若毛利率 ≥ 52.7% → F1 未触发
```

**认知危险**：
- ⚠️ 不要被"毛利率下降但绝对利润增长"叙事误导——F1只看相对数字，不接受任何解释性叙事
- ⚠️ -2.9% ≠ 未触发（接近触发线需要更高警觉）

#### 2.2 F2 = WeChat MAU同比（权重 15-25%）

**标准**：
- 基准线：14.18亿（Q4 2025 MAU）
- 触发条件：WeChat MAU < 13.90亿（同比下降超过2%）
- 数据来源：腾讯季度财报用户数据披露

**收集操作**：
```
WeChat MAU同比变化 = (当前MAU ÷ 去年同期MAU - 1) × 100%
若同比下降 > 2% → F2 触发
若同比下降 ≤ 2% → F2 未触发
```

**认知危险**：
- ⚠️ MAU绝对值创新高 ≠ F2未触发——必须看同比变化
- ⚠️ DAU比MAU更重要（若两者披露，优先看DAU）

#### 2.3 F3 = 视频号时长环比（权重 10-15%）

**标准**：
- 基准线：视频号时长同比+20%以上（Q4 2025实测）
- 触发条件：视频号时长同比 < +10%（增速腰斩）
- 数据来源：腾讯财报管理层讨论 / QuestMobile等第三方数据

**收集操作**：
```
视频号时长同比 = (当前时长指标 ÷ 去年同期 - 1) × 100%
若同比增速 < +10% → F3 触发
若同比增速 ≥ +10% → F3 未触发
```

**认知危险**：
- ⚠️ F3权重最低，不应单独作为卖出依据
- ⚠️ 若F1和F2未触发，F3触发只作为"降低仓位优先级"信号

#### 2.4 竞争对手数据（权重影响，不改变标准）

**收集时机**：5月7-10日（阿里/百度/京东Q1发布期间）

| 公司 | 发布日 | 采集重点 | 对F1/F2/F3的含义 |
|------|-------|---------|----------------|
| 阿里巴巴 | 5月7-8日 | 广告收入增速/淘天MAU | 广告市场收缩→腾讯F1压力加重 |
| 百度 | 5月8-9日 | 核心广告收入/_margin | 广告市场收缩→腾讯F1压力加重 |
| 京东 | 5月9-10日 | 零售收入/用户数 | 消费环境→F1权重解读参考 |

**权重调整规则（预先声明）**：
- 竞争对手数据**加强**原有F1/F2/F3信号（不是引入新标准）
- 竞争对手数据不能让未触发的F1/F2/F3变成"触发"
- 竞争对手数据不能让已触发的F1/F2/F3变成"未触发"

### Step 3：Inversion判断（独立于Step 2）

**在收集F1/F2/F3数据后，执行Inversion检验**：

**五问**（每问必须回答，不接受"无法判断"）：
1. "如果这个季度F1/F2/F3数据触发，而我没有减仓，5年后我会后悔吗？" → 是 → 纪律执行
2. "如果这个季度F1/F2/F3未触发，但我减仓了，1年后我会后悔吗？" → 是 → 纪律持有
3. "如果我是李录/Buffett，我会因为市场情绪而改变F1/F2/F3标准吗？" → 否 → 不接受叙事
4. "5年后回头看，今天的F1/F2/F3数据有多重要？" → 用于校准决策权重
5. "如果对手在做同样的持仓决定，我会怎么评价？" →芒格伦理可逆性测试

## 三、关键原则（操作层面的绝对纪律）

### 纪律1：F1/F2/F3标准不因叙事而改变

**规则**：不接受任何以下类型的叙事：
- "毛利率下降但利润增长，说明商业模式韧性更强"（不接受）
- "MAU下降是季节性因素，不影响长期价值"（不接受）
- "视频号还在投入期，亏损是战略性的"（不接受）
- "腾讯有微信作为护城河，不会触发F1/F2/F3"（不接受）

**原因**：Morris Ch12 证伪纪律——标准必须在检验之前预先声明，不接受事后解释

### 纪律2：竞争数据改变权重，不改变标准

**规则**：竞争对手数据影响的是"权重解读"，不是F1/F2/F3的量化标准

**示例**：
- 阿里Q1广告收入下滑30% → F1权重解读：行业压力加大，腾讯F1更需严格审视
- 但F1触发标准不改变（仍然是 <52.7%）

### 纪律3：机会成本纪律（Morris Iron Law）

**触发条件**：F1/F2/F3均未触发，但发现明确的更好机会
- 更好机会必须是"能力圈内"的机会
- 更好机会必须有具体的量化支撑（F1/F2/F3标准同等对待）
- "更好机会"不是"看起来可能会涨"——必须是具体的能力圈判断

### 纪律4：Buffett 1969年教训

**核心洞见**：Buffett 1969年不是因为"市场要崩了"而退出，而是因为"找不到符合Graham标准的投资机会"而退出。

**含义**：清仓中煤/移动/盘江不是因为"它们会跌"，而是因为"它们在框架升级后不再是能力圈内的最优选择"——这是机会成本重估，不是市场预测。

## 四、常见陷阱与防御

### 陷阱1："F1接近触发线，但没有完全触发"

**防御**：
- 52.7%以下任何数字都是触发——-2.9ppt ≠ 安全
- 52.7%附近需要更高频监控（下一次财报前月度检查）
- 记录"边缘状态"：下一次若继续恶化，是否会触发？

### 陷阱2："F2下降但F1很好"

**防御**：
- F1权重最高，但F2触发是科技股最重要的早期预警信号
- IBM 2011案例：F2（科技转型）比F1先触发边缘信号
- F2触发即使F1未触发，也需要立即重新评估

### 陷阱3："竞争对手都在跌，所以腾讯跌是正常的"

**防御**：
- 这是社会认同倾向（C1）在起作用
- 用Inversion："如果我今天没有腾讯持仓，我会因为'市场都在跌'而买入腾讯吗？" → 答案显然是否定的
- 竞争对手下跌改变的是F1权重，不改变F1标准

### 陷阱4："这是我等待了很久的机会，我不想错过"

**防御**：
- 这是禀赋效应（G2）和恐惧错过（FOMO）在起作用
- 用Inversion："如果我今天刚研究完腾讯，我会因为'已经等了这么久'而买入吗？"
- Morris「For Now」= 等待的纪律，不是等待的焦虑
