▎指标1：周期中位数 ROIC
公式：ROIC = NOPAT / (股东权益 + 有息负债 - 现金)
 NOPAT = EBIT × (1 - 有效税率)
窗口（按行业刚性匹配）：
 - 煤炭/油气/化工/航运 → 7-10年
 - 公用事业/水务 → 7-10年
 - 消费品牌/医疗器械 → 7-10年
 - 制造（家电/纺服）→ 5-7年
 - 科技平台 → 5年
取「中位数」非均值，避免周期顶/底干扰。
判定：≥25%优秀 / ≥15%好生意 / <15%需特殊理由
要求：逐年列出ROIC明细，再标注中位数。

▎指标2：巴菲特 $1 检验
公式：留存1元创造市值 = (期末市值 - 期初市值 + 区间累计分红 + 区间累计回购) / 区间累计留存收益
其中：区间累计留存收益 = Σ(归母净利润 - 分红 - 回购)
窗口：默认5年；若中间有融资/再融资，剔除融资额。
判定：≥2x优秀 / ≥1x合格 / <1x不合格
要求：列出区间起止市值、累计分红、累计回购、累计留存收益的明细。

▎指标3：FCF Yield
公式：FCF Yield = TTM自由现金流 / 当前市值
 自由现金流 = 经营性现金流 - 资本开支
对水务/管网类：CapEx须用「维护性CapEx」估算（≈D&A）；扩张性CapEx单独列出
对水电类：建成期后CapEx应趋近D&A，新建机组单独列出
判定逻辑：
 - A类：要求 FCF Yield ≥ 股息率 × 1.2（覆盖分红还有冗余）
 - B类：单看Yield无意义，须配合「FCF增速」一起看，给出未来3年FCF CAGR预测

▎指标4：李录置信度
两个置信度独立给出，不要混合：
 - 2年置信度：未来2年公司不发生重大经营退步的概率（≥90%为B类门槛）
 - 10年置信度：10年后公司仍是其行业核心玩家、ROIC不显著下台阶的概率（≥70%为B类门槛）
要求：每个置信度必须列出「支撑因素」和「证伪情景」各2-3条，禁止只给数字不给推理。

▎指标5：Exit PE / IRR 反推买点（仅B类）
正向估算：
 目标市值 = N年后净利润 × Exit PE
 目标股价 = 目标市值 / 当前股本（假设股本不变；若有回购则注明摊薄假设）
反推买点：
 合理买入价 = 目标股价 / (1 + 目标IRR)^N
分层买点（7档分层，下层权重更高）：
 以「合理买入价」为基准价P₀
 Tranche 1: 1.15 × P₀ (5%资金)
 Tranche 2: 1.07 × P₀ (8%)
 Tranche 3: 1.00 × P₀ (12%)
 Tranche 4: 0.92 × P₀ (15%)
 Tranche 5: 0.85 × P₀ (20%)
 Tranche 6: 0.75 × P₀ (20%)
 Tranche 7: 0.62 × P₀ (20%)
压力测试：Exit PE 砍至基准的 2/3（如腾讯 15x → 10x）时的合理买入价，作为「绝对底线」参考，非base case。
