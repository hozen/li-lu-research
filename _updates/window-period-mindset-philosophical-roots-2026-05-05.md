## 一、Hume（1711-1776）：归纳问题的起源

### 1.1 核心命题

David Hume 的**归纳问题**（The Problem of Induction）是西方哲学最重要的认识论问题之一：

> 从"过去一直如此"推论"未来也会如此"——这个推论在逻辑上**无法被证明**。

**例子**：
- 太阳每天升起 → 但不能证明"明天一定升起"（只是习惯性期待）
- 水在100°C沸腾 → 不能证明"下一次水在100°C一定沸腾"（可能下次环境不同）
- "这只股票过去10年每年涨10%" → 不能证明"未来10年每年也涨10%"

### 1.2 Hume 的解释

我们之所以用归纳法，**不是因为它逻辑上可靠**，而是因为它**心理上不可避免**——人类天生通过重复经验形成习惯。Hume 说：我们不是通过理性得出归纳结论，而是通过**习惯**（custom）将过去的经验投射到未来。

### 1.3 对投资的含义

| 常见投资陈述 | Hume 问题 |
|-------------|---------|
| "这只股票过去10年每年涨10%" | 不能证明"未来也会" |
| "经济一直从衰退中复苏" | 下次不一定 |
| "价值投资长期有效" | 无法从历史归纳中证明 |
| "腾讯护城河很强所以继续持有" | 强护城河不等于永续 |

**Graham 的回应**：margin of safety = 在 Hume 问题面前构建安全结构。在不可靠的归纳推理周围建立缓冲，使错误不会导致灾难性损失。

## 二、Reichenbach / Popper（1930s-1950s）：实用主义归纳与证伪主义

### 2.1 Reichenbach：最佳解释推理

Hans Reichenbach 提出了**实用主义归纳**（Pragmatic Induction）：
- 归纳法虽然无法在逻辑上保证，但可以通过**最佳解释推理**（Inference to the Best Explanation, IBE）来赋予它概率基础
- 不追求绝对确定，只追求"在当前信息下最可能正确的判断"

**对投资的含义**：我们不需要"证明"腾讯会继续成功，只需要"在现有信息下，腾讯继续成功的概率最高"。

### 2.2 Popper：证伪主义

Karl Popper 在 Reichenbach 基础上提出**证伪主义**（Falsificationism）：

**核心命题**：
1. 科学理论无法被"验证"，只能被"证伪"
2. 科学进步的机制：提出猜想 → 尝试反驳 → 幸存下来的理论获得了更高的确证度
3. **证伪条件必须在检验之前预先声明，不能在检验之后更改标准**

**Popper 三规则**：
- **规则一（预先声明）**：证伪条件必须在检验前声明
- **规则二（不事后更改）**：标准一旦确定，过程中不得更改
- **规则三（不对称性）**：验证无法确立真理，单次证伪即可推翻

**对 Morris「For Now」的直接影响**：
- Morris「For Now」F1/F2/F3 条件 = 预先声明的证伪条件
- Popper 的"不能事后更改标准" = Morris「For Now」的纪律核心
- 腾讯Q1财报（5月13日）= Popper 意义上的"证伪检验"

## 三、Wilfrid Sellars（1950s-1960s）：两种知识图景

### 3.1 核心命题

Sellars 提出了**"两种知识图景"**（Two Images of Knowledge）的对立：

| | 第一种图景（The Way of Given） | 第二种图景（The Framework View） |
|---|---|---|
| 知识来源 | 直接经验的记录 | 语言和实践框架中的社会角色 |
| 哲学传统 | 经验主义 | 概念主义/社会建构主义 |
| 投资表现 | "我看到股价下跌了，所以是风险" | "我在一个特定的概念框架内评估这个下跌" |

### 3.2 对投资的含义

**第一种图景投资者**：
- "腾讯股价跌了 → 风险提高了 → 卖出"（直接感知驱动）
- 问题：市场短期波动 ≠ 基本面变化，这是 Hume 归纳问题的典型误用

**第二种图景投资者**：
- "腾讯股价跌了，但 F1/F2/F3 未触发 → 框架判断：继续持有"（理论驱动）
- Morris「For Now」= 典型的第二种图景投资者（F1/F2/F3 = 概念框架，不等于直接感知）

**关键洞见**：Sellars 的第二种图景为 Morris「For Now」提供了哲学合法性——在框架内评估，不是认知偏见，而是哲学上正当的认识论立场。

## 四、Morris（2000s）：投资中的实用主义认识论

### 4.1 核心框架

Morris 的实用主义认识论对投资有三个核心贡献：

**① 时间维度**
- 投资中的归纳问题不是"是否"的问题，而是"多快"的问题
- "腾讯的竞争优势会持续" → 不是"是/否"，而是"在什么时间范围内"
- Morris「For Now」的10-30年窗口期 = 对 Hume 问题的时间边界声明

**② 权重维度**
- 不同证据有不同的认知权重
- F1（FBS Margin）> F2/F3 → Morris SDI 权重机制的历史认识论来源
- 不是所有证伪证据等权，而是根据 SDI 维度有优先级

**③ 制度维度**
- 归纳问题在有制度保障的情况下有不同的概率结构
- Morris Ch14 的制度修正机制 = 制度对归纳推理概率结构的改善
- China 2.5 的含义：在有制度保障的情况下，Hume 问题可以更乐观地处理

### 4.2 Morris「For Now」的认识论定位

Morris「For Now」= 将 Popper 证伪主义 + Reichenbach 最佳解释推理 + Morris SDI 权重机制整合为一个可执行的投资纪律系统：

| Popper 提供 | Morris「For Now」对应 |
|------------|-------------------|
| 预先声明证伪条件 | F1/F2/F3 精确量化标准 |
| 不能事后更改标准 | Morris Ch12 规则二 |
| 单次证伪 > 无数次验证 | F1 权重 >25% |
| 猜想 → 反驳循环 | 每季度证伪检验 |

## 六、窗口期心态纪律的思想史总结

**窗口期心态纪律（"不焦虑/不预测/不行动/持续监控"）的哲学根源**：

### 6.1 不焦虑 → 来自 Hume

归纳问题结构上是不可解决的。焦虑没有用，只有纪律有用。
- 股价每天波动 → 是 Hume 归纳问题的日常体现
- 我们无法"证明"明天会怎样 → 因此焦虑在认识论上没有根基
- 正确回应：建立预设的纪律框架（F1/F2/F3），而不是试图预测

### 6.2 不预测 → 来自 Popper

预测无法被证伪，只有检验预先声明的条件才有意义。
- "我认为腾讯会涨到X元" → 这是无法证伪的猜想，不是投资纪律
- "如果 F1 触发，则执行卖出" → 这是可以被证伪的条件句
- Popper 的教训：**可证伪的陈述才是科学陈述，可证伪的投资规则才是投资纪律**

### 6.3 不行动 → 来自 Morris「For Now」

在 F1/F2/F3 未触发时，没有行动的技术理由。
- Popper 的不对称性：没有证伪证据 ≠ 证实
- Morris「For Now」精确表述："F1/F2/F3 未触发" = 没有证伪证据 ≠ 腾讯必然继续成功
- 因此：**不行动不是因为我们确定腾讯会成功，而是因为没有触发卖出条件**

### 6.4 持续监控 → 来自 Morris SDI 权重机制

F1/F2/F3 是高频代理指标，需要持续跟踪。
- Morris SDI 的权重机制要求持续监控 Energy Capture 维度的变化
- 窗口期心态纪律的"监控"不是焦虑地看股价，而是**定期检验预设的证伪条件**

## 七、Morris「For Now」窗口期的终极哲学含义

### 7.1 不是躺平，而是哲学上的诚实

Morris「For Now」不是"躺平什么都不做"，而是：

> **在 Hume 归纳问题面前，通过 Popper 证伪主义和 Morris SDI 维度，构建了一个有纪律的认知和行动框架。**

窗口期心态纪律 = 哲学上的诚实（承认归纳问题的不可解决性）+ 操作上的纪律（预设条件触发才行动）

### 7.2 窗口期心态纪律的四个层次

| 层次 | 来源 | 内容 |
|------|------|------|
| 认识论层 | Hume | 归纳问题不可解决，只能管理 |
| 方法论层 | Popper | 预先声明条件，不能事后更改 |
| 框架层 | Morris SDI | F1/F2/F3 = 高频代理指标 |
| 操作层 | Morris「For Now」 | 不焦虑/不预测/不行动/持续监控 |

### 7.3 与 Buffett 纪律的关系

Buffett 的纪律（"在别人恐惧时贪婪，在别人贪婪时恐惧"）如果没有 Morris「For Now」的框架，会变成：
- 主观判断（什么是"恐惧"？谁来判断？）
- 容易被事后合理化（"我认为是恐惧时刻，所以我买入"）

Morris「For Now」将 Buffett 的格言**操作化**：
- "别人恐惧时贪婪" → 如果 F1/F2/F3 未触发，且价格提供更大 Margin of Safety
- "别人贪婪时恐惧" → 如果 F1/F2/F3 触发，或价格不再提供 Margin of Safety

## 八、5月13日腾讯Q1财报的认识论含义

### 8.1 检验之前

- 所有判断都是**猜想**（Conjecture）
- 腾讯持仓 = 一个尚未被检验的猜想

### 8.2 检验之后（5月13日）

- **结果A（F1+F2+F3 全部未触发）** = Popper 意义上的"暂时未能证伪"（不是"验证"！）
- **结果B（F1 触发）** = 认识论上权重最大的单次证伪（Morris 权重修正），需要严肃对待
- **结果C（F2/F3 触发但 F1 未触发）** = 灰色地带（Morris 权重修正精确应用，需要更多判断）

### 8.3 关键认识论区分

| | Popper 验证 | Morris「For Now」不验证 |
|---|---|---|
| 含义 | 证明理论为真 | 无法证明，只能未证伪 |
| 腾讯Q1 | F1/F2/F3 未触发 ≠ 腾讯必然继续成功 | 只是"没有证伪证据" |
