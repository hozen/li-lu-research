## 一、Morris「For Now」核心框架

### 1.1 框架本质

Morris「For Now」= 在Hume归纳问题面前的诚实体认 + 在有限历史窗口内的最大化复利行动

**三层时间结构**：

### 1.2 F1/F2/F3 三重证伪条件

**F1 = FBS毛利率（权重 >25%，能量获取维度）**
- 基准：5.7%（2024Q4实测）
- 触发线：计划削减3ppt → 2.7%
- 本质：护城河能量获取能力的季度代理指标

**F2 = WeChat MAU同比（权重 15-25%，网络效应维度）**
- 基准：14.18亿（2025Q1同比）
- 触发线：计划削减2% → 13.90亿
- 本质：社会组织基础设施（微信）的规模密度

**F3 = 视频号时长环比（权重 10-15%，创新维度）**
- 基准：20%（相对基准期）
- 触发线：计划削减10% → 18%
- 本质：熵减自强化速度的创新代理指标

### 1.3 季度决策矩阵（5月13日执行版本）

- **0/3触发** → 继续持有，窗口期延续
- **1/3触发** → 重新评估（芒格伦理可逆性第九层：政府纠偏意愿）

### 4.1 文明3.0窗口期（1776-2100年）

**Buffett 19.7%年化的历史必然性** = Morris SDI扩张期 × 市场效率缺陷 × 制度保障

**文明三阶段**：
- 1.0采集文明：人均GDP ~0-100美元，无资本市场
- 2.0农业文明（~11500年前-1750年）：马尔萨斯陷阱，始终0-100美元

2. **AI奇点**：通用人工智能实现 → 信息处理维度根本性改变
3. **制度重构**：全球治理体系根本性重写 → Morris Ch14制度信任重写

2026年：Nightfall仍然是预测，不是现实。

### 4.3 China 2.5 = Buffett框架在中国的精确有效期

- China 2.0（1978-2010）：M1→M2转型成功验证
