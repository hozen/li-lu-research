一、PTF 函数定义

PTF（Phase Transition Function） 出自 Morris《战争！》（2016），是 Morris SDI 的数学重构。PTF 衡量"给定能量获取和组织能力，一个社会能维持多高的组织复杂性"。

PTF(t) = N(t) x e(t) x A(t) x mu(t) # N(t) = 人口规模（人口数量） # e(t) = 人均能量获取（每人每年能量，J/人/年） # A(t) = 组织复杂性（制度/社会结构 x U*(t)权重） # mu(t) = 知识效率（单位知识获取成本的倒数，认知效率）

ECoE（Energy Cost of Energy）与 PTF 的关系

ECoE = 获取一单位可用能量所需的能量单位。ECoE 与 PTF 呈反向关系：

# ECoE(t) = 1 / [e(t) x mu(t)] (近似) # ECoE 越低 -> 净剩余能量越多 -> PTF 越高 # ECoE 越高 -> 净剩余能量越少 -> PTF 越低

PTF 是乘法结构：任意一项为零或负，PTF 即崩溃。这解释了为什么文明崩溃往往是非线性的（乘法效应），而非线性的逐步衰退。

二、PTF lambda（增长率）分解

PTF(t) = N x e x A x mu lambda(t) = dPTF/dt / PTF = lambda_N + lambda_e + lambda_A + lambda_mu # lambda_N = dN/dt / N (人口增长率) # lambda_e = de/dt / e (人均能量变化率) # lambda_A = dA/dt / A (组织复杂性变化率) # lambda_mu = dmu/dt / mu (知识效率变化率)

三、Morris For Now 的 sigmoid 定位

lambda(t) = lambda_max / (1 + exp(-k*(t-t_mid))) # lambda_max ~ 0.050 (1850-1950 年峰值) # t_mid ~ 2050 (Morris For Now 中点) # k ~ 0.07 (sigmoid 陡度参数) # 当前 2026: t = t_mid - 24年, lambda ~ 0.029

sigmoid 四性质

  • lambda 一阶导：变化率的变化（加速度），sigmoid 两侧缓慢，中段剧烈
  • lambda 二阶导：t_mid 附近 dlambda/dt 最大
  • 极限行为：t趋向负无穷 lambda趋向0；t趋向正无穷 lambda趋向lambda_max
  • 中点特殊性：t_mid 处 dlambda/dt = 0

China 2.5 vs Buffett 1974 的根本差异：1974 年 lambda(能源) 从正骤降至 -0.300，是单一 sigmoid 变量骤变。China 2.5 当前 HC1+HC4 双轨均为正向，结构性条件根本不同。

四、HC4（DeepSeek R1）与 PTF 第四项 mu

HC4（DeepSeek R1）= PTF 第四项 mu（知识效率）的大幅改善。mu = 单位知识获取成本的倒数（认知效率）。

HC4 的五层传导链

  1. AI 辅助科研 -> 知识积累速度加快 -> lambda_mu 正向
  2. AI 优化能源调度 -> ECoE(能量) 下降 -> lambda_e 正向
  3. AI 提升组织效率 -> A(制度) 效率提升 -> lambda_A 正向
  4. AI 增强数字平台 -> Ostrom 退出选项扩大 -> F3 晴雨表正向
  5. AI 降低认知成本 -> 认知 ECoE 下降 -> 护城河维护成本下降 -> Buffett 护城河存续期延长

DeepSeek R1 不是在解决能量问题，而是在解决组织/制度效率问题——这使它成为 F2 晴雨表（政治经济学层）的正向输入，而不是 HC1（能量层）的替代品。
